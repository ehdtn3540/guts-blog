vim 환경에서 편집 작업을하고 저장을 위해 매번 한영전환을 해주는일은 여간 귀찮은 일이 아니다.  
이때 im-select라는 vim 플러그인을 사용하여 Esc 입력시 영문으로 자동 전환 되도록 해준다.  
참고: 아래 내용은 Mac OS(M1), 순정 vim 환경을 기준으로 작성되었다.

---

## Homebrew를 통한 im-select 설치
**im-select 설치**
```
brew tap daipeihust/tap # im-select 설치를 위한 개인 저장소 등록
brew install im-select # im-select 설치
```
**오류발생시**
```
# 오류 메시지: 개인이 만든 저장소를 불러올때 '신뢰' 상태로 변경해야 설치 가능
Error: Refusing to load formula daipeihust/tap/im-select from untrusted tap daipeihust/tap.
Run `brew trust --formula daipeihust/tap/im-select` or `brew trust daipeihust/tap` to trust it.

# 저장소를 신뢰 상태로 변경
brew trust daipeihust/tap
```
**설치 경로 확인**
```
# /opt/homebrew/bin/im-select -> 이 경로가 나오면 정상 설치
which im-select
```

## Vim 설정 파일 수정 
```
# 설정 파일 열기
vim ~/.vimrc

" 키 입력 반응 속도 최적화 (딜레이 제거 핵심 설정)
set timeoutlen=1000  " 일반 단축키 조합 대기 시간 (1초)
set ttimeoutlen=10   " Esc 키 입력 반응 속도 제한 (10ms로 단축)

" Vim 환경에서 Esc를 누를 때 영문(ABC) 자판으로 자동 전환
autocmd InsertLeave * silent !/opt/homebrew/bin/im-select com.apple.keylayout.ABC

" Vim을 완전히 종료하고 터미널로 나갈 때 영문 자판으로 자동 전환
autocmd VimLeave * silent !/opt/homebrew/bin/im-select com.apple.keylayout.ABC
```
