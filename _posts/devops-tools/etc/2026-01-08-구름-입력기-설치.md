시작은 iterm2 환경에서 한글입력후 삭제시 조합상태에 걸려있어
삭제(백스페이스)를 두번 눌러야 삭제되는 현상이 발생하여 방법을 찾다가 
구름 입력기라는것을 발견하여 해결했다. 아래는 구름 입력기 설치 및 사용 방법이다.  
환경: MacOS, iterm2

## 구름 입력기 설치 방법
1. 구름 입력기 공식 사이트 접속
https://gureum.io
2. MacOS 용 구름 입력기 설치


## 구름 입력기 사용 방법
1. MacOS 시스템 설정  

- 구름 입력기 두벌식 추가  
시스템 설정 -> 키보드 -> 텍스트 입력 -> (입력 소스)편집  
 -> 좌측 하단 "+" 클릭 -> 한국어 -> (구름 아이콘)두벌식 -> 추가  
 참고: (구름 아이콘)두벌식이 보이지 않을 경우 컴퓨터 재부팅후 다시 시도  
- 충돌 방지를 위한 기존 입력기 삭제  
편집에서 기존 두벌식 클릭 -> 좌측 하단 "-" 클릭


## 구름 입력기가 좋은 이유
- 입력 반응이 빠름: 타이핑 지연이 적고 키 입력이 즉각적  
- 조합 정확도 높음: 오타·중복 입력이 적음  
- 키 커스터마이징 가능: 한/영 전환, Caps Lock 등 자유도 높음  
- 오픈소스: 신뢰도 높고 지속적인 개선  
- 개발자·파워유저 선호: 터미널, 에디터, IDE와 궁합 좋음  
- 불필요한 자동 보정 없음: 입력한 대로 나와서 스트레스 적음
