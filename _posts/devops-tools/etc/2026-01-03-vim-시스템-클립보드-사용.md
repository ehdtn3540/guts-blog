iTerm2 분할창 사이에서 Vim의 y(yank)와 p(put)로 내용을 공유하려면,  
Vim 내부 저장소(Register)가 아닌 macOS 시스템 클립보드를 사용하도록 설정해야 합니다.  
Mac iTerm2 환경에서 2026년 현재 가장 확실한 방법 3가지를 정리해 드립니다.  
---

1. .vimrc 설정 (가장 추천)  

Vim에서 복사하는 모든 내용을 시스템 클립보드로 자동으로 보내는 설정입니다.
이 설정을 하면 A창 Vim에서 y를 누르고, B창 Vim에서 p만 눌러도 바로 붙여넣기가 됩니다.  

```
터미널에서 설정 파일을 엽니다: vi ~/.vimrc  
```

2. 아래 내용을 추가하고 저장(:wq)하세요.  

```
# 시스템 클립보드와 연동 (unnamedplus가 작동하지 않으면 unnamed 사용)
set clipboard=unnamedplus,unnamed
```

이제 어느 창에서든 y로 복사하면 다른 창 Vim이나 카카오톡, 메모장 등에도 바로 Cmd + V로 붙여넣을 수 있습니다.
