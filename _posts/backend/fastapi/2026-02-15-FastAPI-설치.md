## FastAPI란
FastAPI는 Python 기반의 초고속 웹 프레임 워크로  
API 서버를 만들때 가장 많이 사용되고 있다.  

특징으로는
- 빠름 (uvicorn + 비동기 구조 → Node.js 수준 속도)
- 간단함 (Python 함수로 API 만들기)
- 자동 문서화 (Swagger UI 자동 생성)
- 타입 힌트를 강제하지 않지만 지원 → 개발 편함
- SaaS·웹서비스·백엔드 API 만들 때 정말 많이 쓰임.

## 설치방법
```
pip install fastapi uvicorn
```
- fastapi: 프레임워크 본체  
- uvicorn: FastAPI 앱을 실행할 초고속 ASGI 서버  

### Uvicorn에 대해서
Uvicorn(유비콘)은 작성한 코드를 실제로 웹에서  
동작하게 해주는 비동기 웹 서버(ASGI) 서버 입니다.

1. 웹서버 역할(엔진)
FastAPI는 비즈니스 로직을 정의하는 '프레임워크'일 뿐, 스스로 네트워크 요청을 받을  
수는 없기 때문에 Ucicorn을 통해 클라이언트(브라우저 등)의 HTTP 요청을 받아 FastAPI로  
전달하고, 처리된 결과를 다시 클라이언트에게 응답으로 보내주는 실제 실행 엔진 역할을 한다.  

2. ASGI(Asynchronous Server Gateway Interface)
비동기 서버 게이트웨이 인터페이스로, 비동기 방식을 지원하여 HTTP, WebSocket 등  
다양한 요청을 동시에 끊김없이 처리해주는 표준 규격이다.

3. 개발 편의성 
개발 중 코드를 수정하면 서버를 수동으로 껐다 켤 필요 없이,  
자동으로 변경 사항을 감지하여 재시작해주는 --reload 옵션을 제공한다.
