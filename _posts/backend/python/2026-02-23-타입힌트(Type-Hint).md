파이썬의 타입 힌트(Type Hint)는 변수나 함수에 "이 데이터는 이런 종류입니다."라고  
이름표를 붙여주는 것이다. 

---

## 1. 기본 사용법
가장 기본은 변수 이름 뒤에 콜론(:)을 찍고 타입을 적는 것이다.
```
# 1. 변수에 이름표 붙이기
age: int = 25          # age는 정수(int)
name: str = "Alice"    # name은 문자열(str)
is_rich: bool = True   # is_rich는 불리언(bool)
```

## 2. 함수에서 사용법
함수에서는 "무엇을 받고(매개변수), 무엇을 주는지(반환값)"를 명확히 한다.  
화살표(->)를 사용해 결과값의 타입을 표시한다.  
```
# "name은 문자열로 받고, 결과값도 문자열로 줄게"라는 뜻
def say_hello(name: str) -> str:
    return f"Hello, {name}"
```

## 3. 실무에서 자주 사용하는 방법
리스트나 딕셔너리처럼 내부 구성 요소가 있는 경우  
typing 모듈을 가져와서 더 구체적으로 적어준다. 
- List[int]: "정수들이 들어있는 리스트야"
- Dict[str, int]: "키는 문자열이고 값은 정수인 딕셔너리야"
- Optional[str]: "문자열일 수도 있고, 비어있을(None) 수도 있어"

```
from typing import List, Optional

def process_scores(scores: List[int], memo: Optional[str] = None):
    # scores는 숫자 리스트여야 하고, memo는 문자열이거나 없을 수 있음
    pass
```

## 4. 타입힌트를 사용하는 이유
1. 실수 방지: 숫자가 들어가야 할 곳에 문자를 넣으면 에디터가 빨간 줄로 경고를 줍니다.
2. 자동 완성 대잔치: 타입을 적어두면 VS Code 같은 에디터가 그 타입에 맞는 함수(split, append 등)를 알아서 띄워줍니다. (공부할 때 매우 편리!)
3. 의사소통: 코드를 나중에 다시 볼 때 data가 리스트인지, 문자열인지 고민할 필요가 없습니다.
