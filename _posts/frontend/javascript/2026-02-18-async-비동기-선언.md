async는 JavaScript에서 비동기 함수(Asynchronous Funcion)를 정의하는 문법이다.  
쉽게 설명하면 "이 함수는 실행 도중에 멈춰서 기다려야 하는 작업(예: 서버에서 데이터 가져오기)이 포함되어 있다"라고 선언하는 것이다.

---
## 1. async의 의미
- 비동기 선언: 함수 앞에 async를 붙이면, 이 함수는 항상 Promise라는 객체를 반환한다.
- await: 이 함수 내부에서는 await라는 키워드를 사용할 수 있게 된다.

## 2. await의 역할(짝꿍 키워드)
보통 함수 내부의 fetch 코드 앞에 await가 붙어 있다.
```
const response = await fetch("http://localhost:8000/api/test");
```
보통 자바스크립트는 코드를 한 줄씩 빠른 속도로 실행하고 바로 다음 줄로 넘어간다.  
하지만 서버에 데이터를 요청하면 응답이 올 때까지 시간이 걸린다.  
이때 await는 "서버에서 응답이 올 때까지 다음 줄로 넘어가지 말고 여기서 잠깐 기다려!"라고  
명령하는 역할을 한다. 만약 await가 없다면, 데이터가 채 오기도 전에 다음 코드가 실행되어  
에러가 발생한다.


## 3. 이렇게 쓰는 이유
과거에는 비동기 처리를 위해 clallback 함수나 .then()을 길게 늘어뜨려 사용했는데,  
코드가 매우 복잡해지는 단점이 있었다.(콜백 지옥) 그러나 async/await를 사용하면
1. 비동기 코드를 일반적인 동기 코드(위에서 아래로 흐르는 코드)처럼 읽기 쉽게 작성할 수 있고
2. try...catch 구문을 사용해 에러 처리를 직관적이로 할 수 있다.
