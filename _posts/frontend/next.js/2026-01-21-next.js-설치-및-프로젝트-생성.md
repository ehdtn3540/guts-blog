next.js 프로젝트 생성을 위한 node.js 설치 및 next.js 프로젝트 생성 및 테스트  
참고: MacOS 환경을 기준으로 한다.

---

## node.js 설치
```
brew install node # node.js 설치
node -v # node.js 설치 확인 
npm -v # npm 패키지 설치 확인
```

## Next.js 프로젝트 생성 및 서버 실행
```
cd ~/Documents # 프로젝트 설치 위치로 이동
npx create-next-app@latest [프로젝트명] # 설치 옵션에서 대부분 Enter를 눌러 기본값 사용

cd [프로젝트명] # 프로젝트로 이동
npm run dev # 서버 실행, http://localhost:3000 문구 뜨면 성공
```

## 작동 테스트 및 수정
1. http://localhost:3000에 접속하여 Next.js 기본 페이지가 나오는지 확인
2. app/page.tsx 파일을 열어 기본 페이지 내용 수정
```
export default function Page() {
  return <h1>내 맥에서 돌아가는 Next.js!</h1>
}
```
3. 수정 내용 저장 후 브라우저 서버 접속해서 변경 내용 적용 확인
