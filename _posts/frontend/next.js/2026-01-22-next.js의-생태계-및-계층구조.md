## 계층 구조(큰 개념부터)
### Node.js
Next.js의 가장 밑바닥에 깔리는 엔진으로 JavaScript 런타임 환경이다.  
JavaScript를 브라우저 밖(서버)에서 실행할 수 있게 해주는 환경이다.   

Next.js의 서버 사이드 기능(SSR)이나 라우트는 결국 Node.js 서버 위에서 동작하며  
따라서 Node.js가 없으면 Next.js는 실행조차 불가능 하다.  
### React
UI를 만들기 위한 부품(컴포넌트) 조립 도구이지만  
경로 이동(Routing)이나 서버 통신 방식은 정해져 있지 않다.

Next.js의 모든 컴포넌트 문법은 React를 따른다.  
### Next.js
React를 기반으로 하되, 개발에 필요한 라우팅, SSR, 성능 최적화 도구들의 규칙을  
미리 정해둔 완성형 툴이다. 이 모든 도구들을 React 위에 얹어 놓은 형태이다.
