git 명령어 alias(약어)를 설정하여 명령어 타이핑 시간을 줄이고 오타를 줄이고자하는 목적에 있다.
---

## 방법1
git config 명령어를 사용하여 alias 설정하는 방법
```
# status 명령어를 st로 설정
git config --global alias.st status
```

## 방법2
.gitconfig 파일을 직접열어 alias 설정
```
# 설정파일 열기
vi ~/.gitconfig

# alias 추가
[alias]
    st = status
    df = diff
    co = checkout
    br = branch
    cm = commit
    last = log -5 HEAD
```


## 설정 확인
```
# 설정한 alias 명령어를 확인하는 명령어
git config --global --get-regexp alias
```
